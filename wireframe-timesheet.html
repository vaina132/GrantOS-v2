📐 Interactive Wireframe — Daily Timesheet Grid
This is a clickable prototype showing the new timesheet layout. Click cells to enter hours. Key features:
  • Rows = projects (with WP sub-rows). Columns = working days of March 2026.
  • Gray columns = weekends (not editable). Red header = national holiday. Blue cells = absence days.
  • Bottom row = daily totals with capacity bars (green ≤8h, amber ≤8h, red >8h).
  • Right column = monthly project total + PM equivalent.
  • Action buttons: Fill from Plan (auto-distribute), Copy Previous Month, Submit.
